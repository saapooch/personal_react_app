## Get Started
  ### Using a virtual environment
  You'll need the following installed:
  * npm

  To install the dependencies:
  ```
  npm install
  ```
  Run the application on a local server:
  ```
  npm run dev
  ```

  Go to the [application](http://localhost:9000/)
